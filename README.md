# textmode-playground
Playground for console animations (ASCII-art/Unicode-art).

## Requirements
Tested on GNOME Terminal (default Ubuntu Mono 12 font), and braille font support.
- Python-fontconfig (0.5.1)  - Python bindings for Fontconfig library
